---

## 🎯 What is Veronica?

Veronica is an AI-powered productivity companion that **proactively** helps users plan, prioritize, and complete tasks before deadlines are missed — going far beyond passive reminders.

She has a face, a voice, and a personality. She listens, understands context, and helps you *take action*.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🧠 **Gemini AI Brain** | Powered by Gemini 2.0 Flash for intelligent task analysis |
| 👩 **Animated AI Avatar** | Fully animated SVG girl with blinking, breathing, lip-sync |
| 🎙️ **Voice Output** | Veronica speaks her responses using Web Speech API |
| 📋 **Smart Task Manager** | Priority-sorted tasks with AI-auto-extraction from chat |
| 🎭 **Mood System** | 4 focus modes (focused/motivated/calm/urgent) that change the entire UI |
| ⚡ **Autonomous Task Planning** | AI suggests tasks → they appear on your board automatically |
| 📊 **Productivity Insights** | Live productivity score, energy schedule, week heatmap |
| ⏰ **Context-Aware** | Deadline tracking, time display, energy-based scheduling |
